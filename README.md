# muON-DTN
Highly extensible and portable DTN network firmware for microcontroller platforms
Developed as a set of modules atop the GGG framework: https://github.com/albertos642/ggg
# This repository is currently a WIP
The code of muON-DTN is currently under review in order to be cleaned up, thoroughly tested for vulnerabilities and made ready for public access. It will thus be made available shortly. Thank you for your patience.
